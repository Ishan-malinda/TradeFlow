# Foundations Images Folder

## How to Add Images

1. **Place your images in this folder** (`public/images/foundations/`)
   
2. **Supported formats**: `.jpg`, `.jpeg`, `.png`, `.svg`, `.webp`

3. **Recommended image names** (you can use any names you prefer):
   - `intro-trading.jpg` - Introduction to Trading
   - `financial-markets.jpg` - Financial Markets
   - `market-participants.jpg` - Market Participants
   - `market-types.jpg` - Different Market Types
   - `brokers.jpg` - Brokers
   - `prop-firms.jpg` - Prop Firms
   - `trading-platforms.jpg` - Trading Platforms
   - `trading-roadmap.jpg` - Trading Roadmap
   - `trader-mindset.jpg` - Trader Mindset

4. **Image sizes**: 
   - Recommended width: 900-1200px
   - Recommended format: JPG or PNG
   - Keep file sizes under 500KB for faster loading

5. **After adding images**, they will automatically appear on the Foundations page!

## Example

If you add an image named `intro-trading.jpg` to this folder, it will be accessible at:
`/images/foundations/intro-trading.jpg`

The Foundations page is already configured to use these image paths.
